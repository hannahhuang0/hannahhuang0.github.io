Pattern Blocks

Specifications

Materials: basswood, frosted acrylic, polymer clay, arduino, and circuitry

Size: 16cm x 11.5cm x 6cm

Pattern Blocks is a tangible interface that explores light, diffusion, patterns, and design, encouraging experimentation and creativity. Using building blocks and light, it allows the user to form unique designs and different patterns. It helps the users, such as children, to practice their problem-solving and creative thinking skills while engaging in design. This device consists of a wooden box, frosted acrylic cubes that rest on top of an acrylic base, and polymer clay buttons.
