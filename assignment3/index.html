Typography  Poster  Designer

HOW TO USE

  • Min canvas size: 100 × 100px
    (w × h)
  • Max canvas size: 900 × 900px
    (w × h)
  • Enter: enter/update text input
  • Delete: remove selected text
  • Click: select, translate, rotate text
  • Double-click text: edit selected text
  • F2: fullscreen mode
  • Esc: exit fullscreen mode
  • Tip: text can be used to form lines and shapes
